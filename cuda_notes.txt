Getting CUDA working is a prerequisite for using this package.

## What you need

1. An NVIDIA GPU with compute capability >= 3.5.
2. A recent NVIDIA driver. Install via your OS package manager (e.g. `apt install nvidia-driver-XXX`
   on Ubuntu) or from https://www.nvidia.com/drivers. After installation, `nvidia-smi` should
   display your GPU.

That's it. CUDA.jl downloads and manages the CUDA runtime automatically -- no manual CUDA
toolkit installation is required.

## Verifying the setup in Julia

```julia
using CUDA
CUDA.functional()   # should return true
CUDA.versioninfo()  # shows driver, toolkit, and device info
```

## Troubleshooting

- `CUDA.functional()` returns `false`: check that `nvidia-smi` works outside Julia. If not,
  reinstall or update the driver.
- Driver/runtime version mismatch warnings: run `Pkg.update()` to get a compatible CUDA.jl
  version, or update your NVIDIA driver.
